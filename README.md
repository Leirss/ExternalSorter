# ExternalSorter
 SDU数据结构课设测试数据
 
## 使用方法
运行`main.cpp`即可得到生成数据。
阅读代码你会发现数据规模可以不断增加，你可以测试不同规模的数据（不局限于给定）查看你程序的性能表现。

## 外排序任务中输入输出格式

## 输入格式

第一行一个`n`表示文件中有几个元素

```shell
4
233 2 1 10 
```

## 输出格式

```shell
1 2 10 233
```

## 注意！！！

1. 一定应用**最小输者树**设计实现外排序
2. 要体现内外存读取。如：实验指导书中的采用小文件来模拟磁盘块。

## 加分项

对内外存读取实现了更高程度上的模拟可获得更高的分数
